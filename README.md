# AI 开发工作流

1. 利用AI工具（ChatGPT等）将需求结构化

2. 细化补全，从项目结构、技术栈说明到数据库设计都让 AI 辅助完成

3. 将以上过程生成的几个详细markdown文档交给cursor来完成一个项目

# 项目交付清单

1. 源代码

2. 演示视频

3. 环境搭建与代码部署视频

4. 详细的文档（Word / PDF）

5. 环境搭建所需的安装包及其他文件
